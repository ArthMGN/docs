Esta documentação apresenta, de forma objetiva e técnica, as APIs públicas disponibilizadas pelos sistemas da **MGN SYSTEM**, incluindo:

- **MGN SYSTEM - AG CONTROL**

Cada sistema é independente e possui suas próprias integrações, mas todos seguem o mesmo padrão de autenticação, estrutura de payloads e governança multi-tenant, garantindo **segurança, escalabilidade e rastreabilidade operacional**.

<Note>
  Antes de definir quais as integrações serão utilizadas para a operação é necessário verificar como foi ou será realizada a configuração dos usuários no sistema. Qualquer dúvida favor entrar em contato.
</Note>

## Suporte

Dúvidas técnicas ou problemas relacionados às integrações devem ser encaminhados para:

<Icon icon="user-headset" color="#2A3A4A" /> [suporte@mgnsystem.com.br](mailto:suporte@mgnsystem.com.br)

## **Arquitetura**

- APIs expostas via protocolo **REST**
- Formato de troca: **JSON**
- Autenticação: **JWT Bearer Token**
- Suporte multi-empresa (multi-tenant) com base nas permissões do usuário
- Acesso controlado por políticas **RLS (Row-Level Security)**
